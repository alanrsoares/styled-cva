# MCP Docs Server

<Callout type="info" emoji="ðŸ¤–">
  This documentation site functions as an HTTP MCP (Model Context Protocol) server, making styled-cva documentation accessible to AI assistants and other MCP clients.
</Callout>

## Endpoint

**Production URL**: `https://styled-cva.vercel.app/api/mcp`

## Integration

<Tabs items={["Cursor", "Claude Desktop", "Windsurf", "Gemini"]}>
  <Tabs.Tab>

Add to your Cursor MCP configuration:

```json
{
  "mcpServers": {
    "styled-cva-docs": {
      "url": "https://styled-cva.vercel.app/api/mcp"
    }
  }
}
```

  </Tabs.Tab>
  <Tabs.Tab>

Add to your Claude Desktop MCP configuration file (typically `~/Library/Application Support/Claude/claude_desktop_config.json` on macOS):

```json
{
  "mcpServers": {
```

## Available Resources

The server exposes all documentation pages as MCP resources:

- **[Introduction](/)** - `styled-cva://docs/introduction`
- **[Getting Started](/getting-started)** - `styled-cva://docs/getting-started`
- **[API Reference](/api)** - `styled-cva://docs/api`
- **[MCP Docs Server](/mcp)** - `styled-cva://docs/mcp`

## Available Tools

- **`list_docs`** - List all available documentation pages
- **`search_docs`** - Search documentation by keyword (requires `query` parameter)

## `llms.txt`

In addition to the MCP server, we also provide a standard [`llms.txt`](/llms.txt) file. This is a markdown file that provides a concise summary of the project and its documentation, specifically designed for Large Language Models to consume.

You can access it at: `/llms.txt`
